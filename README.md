# langchain-alayalite

`langchain-alayalite` is a LangChain partner package that integrates **AlayaLite** as a vector store backend.

It provides a native `VectorStore` implementation compatible with `langchain-core`, allowing you to use AlayaLite seamlessly within the LangChain ecosystem for similarity search and retrieval.

---

## Installation

Install the package from PyPI:

```bash
pip install langchain-alayalite
